# Installation

## Stable release

To install Montreal_Donnees, run this command in your terminal:

```sh
uv add donnees.montreal.ca
```

Or if you prefer to use `pip`:

```sh
pip install donnees.montreal.ca
```

## From source

The source files for Montreal_Donnees can be downloaded from the [Github repo](https://github.com/6396262-bdeb-qc-ca/donnees.montreal.ca).

You can either clone the public repository:

```sh
git clone https://github.com/6396262-bdeb-qc-ca/donnees.montreal.ca
```

Or download the [tarball](https://github.com/6396262-bdeb-qc-ca/donnees.montreal.ca/tarball/main):

```sh
curl -OJL https://github.com/6396262-bdeb-qc-ca/donnees.montreal.ca/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd donnees.montreal.ca
uv sync
```
